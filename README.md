# aws-sdk
Node app with AWS SDK

# Running app
Add .env file with 
AWS_ACCESS_KEY_ID=<key>
AWS_SECRET_ACCESS=<secret>

Make sure proper access is provided for user/user-group/role.

In terminal run
- `node s3.js`
- `node api.js`
- `node ssm.js`
